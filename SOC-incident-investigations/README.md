# SOC Incident Investigations

This folder contains simulated SOC incident investigations based on log analysis performed using Splunk.

Each investigation includes:

• Detection query 
• Investigation findings 
• MITRE ATT&CK mapping 
• Security impact analysis 
• Incident response recommendations

## Investigations

1. SSH Brute Force Attack
2. Privilege Escalation Activity
3. Suspicious Command Execution
4. Sensitive File Access
5. Log Tampering Attempt

These investigations simulate the workflow of a SOC analyst identifying suspicious activity, analyzing logs, determining attacker behavior, and recommending security response actions.
