# Threat Intelligence Analysis Labs

This folder demonstrates practical threat intelligence workflows including IOC analysis, enrichment, attribution, and reporting.

## Skills Demonstrated
- IOC analysis
- Indicator enrichment
- OSINT research
- Threat actor profiling
- Intelligence reporting

## Topics Covered
- IP reputation analysis
- Domain investigation
- Malware hash lookup
- Campaign tracking
- ATT&CK mapping

## Intelligence Workflow
1. Collect indicators
2. Enrich data
3. Analyze context
4. Assess threat level
5. Report findings

## Outcome
Shows ability to translate raw indicators into actionable intelligence.
