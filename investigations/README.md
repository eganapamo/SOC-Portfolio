# Investigations Lab Series

This folder contains hands-on incident investigation labs demonstrating log analysis, attack tracing, and threat reconstruction skills using Linux system logs.

## Objectives
- Perform forensic log analysis
- Reconstruct attacker timelines
- Identify brute force attempts
- Detect privilege escalation
- Trace lateral movement

## Skills Demonstrated
- grep, awk, sort, uniq pipelines
- Authentication log parsing
- Indicator extraction
- Pattern correlation
- Timeline reconstruction

## Methodology
Each lab follows a structured SOC workflow:

1. Identify suspicious activity
2. Extract relevant data fields
3. Correlate events
4. Validate indicators
5. Document findings
