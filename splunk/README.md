# Splunk Detection & Threat Hunting Labs

This folder demonstrates SIEM-based detection, investigation, and threat hunting using Splunk SPL queries.

## Skills Demonstrated
- SPL search creation
- Alert rule logic
- Dashboard queries
- IOC hunting
- Correlation searches

## Focus Areas
- Authentication anomalies
- Suspicious processes
- Network indicators
- Persistence behavior
- Privilege escalation

## Methodology
1. Define hypothesis
2. Query logs
3. Correlate results
4. Validate indicators
5. Document findings

## Outcome
Shows real SIEM analyst capability and platform proficiency.
